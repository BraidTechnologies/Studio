There's no additional charge to use generative AI tools in Azure Machine Learning. You’ll incur separate charges for compute and for other Azure services such as Azure Blob Storage, Azure Key Vault, Azure Container Registry, and Azure Application Insights when used with Azure Machine Learning. See Azure Machine Learning pricing.                02/       How do I start using Azure Machine Learning?
